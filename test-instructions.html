To test the connection functionality, you need two different browser windows/tabs or two different devices on the same network.

Testing Steps

Step 1: Open Two Browser Windows

Open index.html in two separate browser windows or tabs. This simulates two different users.

Step 2: Allow Location Access

In both windows, allow location access when prompted. This is required for room discovery.

Step 3: Create a Room (User 1)

In the first window:

  1. Click "Create Room"
  2. Enter a room name (e.g., "Test Room")
  3. Set radius to 5 km
  4. Make sure "Allow nearby users to discover this room" is checked
  5. Click "Start Room"

You should see your local video and the room code.

Step 4: Join the Room (User 2)

In the second window:

  1. Click "Join Room"
  2. Enter the room code from User 1
  3. Click "Join Room"

Both users should now see each other's video!

Both users should be connected and see each other's video streams. The connection status should show "Connected! Video should be visible now."

Troubleshooting

If Videos Don't Show:

  • Check Console: Open F12 ‚Üí Console and look for error messages
  • Use Debug Button: Click the "Debug" button in the video chat interface
  • Check Permissions: Ensure camera/microphone permissions are granted
  • Browser Compatibility: Try Chrome or Firefox (Safari may have issues)
  • Network Issues: Make sure both windows are on the same device/network
  • Restart: Try refreshing both windows and starting over

Browser Compatibility

This app works best with:

  • Chrome (recommended)
  • Firefox
  • Edge
  • Safari (may have limitations)
